# Advanced Command-Line To-Do List App

This is an advanced Python command-line To-Do List application. It supports adding, removing, editing, searching, sorting tasks, categorizing, prioritizing, and marking tasks as done for full-featured personal task management.

## Features

- Add, remove, view tasks
- Mark tasks as done and view completed tasks
- Edit existing tasks
- Categories and priorities for each task
- Search tasks by keyword
- Sort tasks by priority (High, Medium, Low)

## Usage

1. **Save the code** to a file (for example, `todo_list.py`).
2. **Run the script** in your terminal:
    ```
    python todo_list.py
    ```
3. **Follow the menu options:**
    - View your current tasks
    - Add a new task with category and priority
    - Edit, remove, or search tasks
    - Mark tasks as done, or view completed tasks
    - Sort tasks by priority for better organization
